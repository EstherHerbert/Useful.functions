# Use the University of Sheffield Colour Palette

`display_unicol()` displays the colour palette in a graphics window.

`get_unicol()` returns the hex colour code for specified colour names
and tints.

## Usage

``` r
display_unicol()

get_unicol(names, tints = 1)
```

## Arguments

- names:

  a character vector of colour names

- tints:

  a real vector of tints, default is 1
