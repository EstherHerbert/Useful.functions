# University of Sheffield Colour Palette

A data frame of the colours in the official Univerisity of Sheffield
colour palette.

## Usage

``` r
unicol
```

## Format

### `unicol`

A data frame with 48 rows and 3 columns:

- Colour.name:

  Colour name

- Tint:

  Where, applicable the tint of the colour.

- Hex:

  Hex colour code
