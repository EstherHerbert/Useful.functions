# Simulated AE data.

A simulated data frame used in examples.

## Usage

``` r
outcome_aes
```

## Format

A data frame with 100 rows and 3 columns:

- screening:

  character, individual id number

- group:

  factor, treatment group A/B

- serious:

  factor with two levels: No, Yes

- related:

  factor with two levels: No, Yes
