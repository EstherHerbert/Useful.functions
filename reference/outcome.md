# Simulated outcome data.

A simulated data frame used in examples.

## Usage

``` r
outcome
```

## Format

A data frame with 600 rows and 11 columns:

- site:

  character, site code

- screening:

  character, individual id number

- group:

  factor, treatment group A/B

- sex:

  factor with three levels: Male, Female, Prefer not to specify

- event_name:

  factor with 3 levels: Baseline, 6 Weeks, 12 Weeks

- pet_dog:

  character, "tick" variable

- pet_cat:

  character, "tick" variable

- pet_fish:

  character, "tick" variable

- score:

  double, randomly generated "score"

- pain:

  factor with 3 levels: Low, Medium, High

- limp_yn:

  factor with 2 levels: Yes, No
